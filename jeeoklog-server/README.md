# JEEOKLOG-SERVER

## 프로젝트 환경
| 기술 | 개발환경 |
| --- | --- |
| Spring Boot | - String Boot 2.7.5 </br> - Java 11 </br> - Gradle |
| Spring Data | - String Data JPA |
| Spring Cloud | - Eureka Client </br> - Config |
| ORM | - JPA </br> - Querydsl |
| Database | - MariaDB |
| Test | - Spring RestDocs </br> - JUnit5 |
